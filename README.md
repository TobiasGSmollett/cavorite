# Cavorite

Cavorite is a work-in-progress actor model library for Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cavorite:
       github: tobiasgsmollett/cavorite
   ```

2. Run `shards install`
